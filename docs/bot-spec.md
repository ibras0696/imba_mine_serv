# Telegram-бот

Бот управляет контейнерами Minecraft через Docker socket и поддерживает несколько серверов.

## Основные возможности

- запуск/остановка/перезапуск выбранного сервера
- статус контейнеров и логи
- просмотр и правка env
- список модов с ссылками
- выбор сервера (main/shooter/vanilla)
- управление мирами (список, переключение, новый мир, загрузка ZIP/URL, бэкапы)

## Настройка

Скопируй `env/.env.bot.example` в `.env.bot` и заполни:
- `BOT_TOKEN`
- `TELEGRAM_ADMINS`
- `WORKDIR`
- список серверов `SERVERS`

## Управление мирами

- **Список миров** — показывает доступные папки с `level.dat`
- **Переключить** — меняет `LEVEL_NAME` и перезапускает сервер
- **Новый мир** — создаёт новый мир со случайным сидом
- **Загрузить мир** — принимает ZIP или ссылку на ZIP (подпись к ZIP = имя мира)
- Команда: `/world_url ИмяМира https://example.com/world.zip`

Бэкапы создаются автоматически (до 3 архивов на сервер).
