# Гайд для игрока (Forge 1.20.1)

Если хочешь подключиться к серверу, делай всё по шагам. Никаких знаний моддинга не требуется.

## 1. Установка Forge клиента
1. Скачай Forge 1.20.1 (тот же билд, что на сервере). Ссылка: https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html  
2. Запусти installer → выбери `Install client` → указать путь до обычного Minecraft.
3. После установки в лаунчере появится профиль `Forge 1.20.1` – именно его запускаем.

## 2. Копирование модов
1. Самый быстрый способ: запусти `bash git/scripts/install_client_mods.sh /путь/к/.minecraft/mods`. Скрипт скачает моды через `make fetch-mods` и скопирует их в указанную папку. (В Windows открой Git Bash, он ставится вместе с Git.)
2. Если хочешь вручную, в репозитории есть папки `mods/server/` и `mods/client/`. Тебе нужны оба набора:
   * **Обязательные** (server+client) – копируй все `.jar` из `mods/server/` в свою папку `.minecraft/mods`.
   * **Клиентские** (оптимизация/JEI и т.д.) – копируй `.jar` из `mods/client/` туда же.
2. Если папки `.minecraft/mods` нет – создай вручную.
3. Никаких других версий модов не ставь (чтобы избежать конфликтов). Если ребята обновляют сборку – просто заменяй файлы на новые.

## 3. Запуск Minecraft
1. Перезапусти лаунчер, выбери профиль `Forge 1.20.1`.
2. Убедись, что выбран DirectX/GL профиль с достаточным объёмом памяти (желательно ≥4 Гб).
3. Нажми `Играть` – в меню появятся моды (Mod List).

## 4. Подключение к серверу
1. Узнай внешний IP и порт сервера (например `123.45.67.89:25565`). Админ должен дать его отдельно.
2. В Minecraft → `Сетевая игра` → `Добавить сервер` → впиши IP:порт.
3. Если видишь сообщение “Server is still starting” – подожди пару секунд и подключись повторно.
4. При любых ошибках смотри лог игры (`.minecraft/logs/latest.log`) или спроси у админа.

## 5. Получение админки (опционально)
Если админ занёс твой ник в переменную `OPS` на сервере, после перезапуска ты получишь права оператора. Иначе – проси его добавить в `OPS=Player1,Player2` и перезапустить сервер.

Готово! Клиентская сборка должна запускаться без дополнительных манипуляций. Если что-то не работает, проверь:
- Версия Forge – строго 1.20.1.
- Все `.jar` из `mods/server` и `mods/client` лежат в твоём `.minecraft/mods`.
- Нет лишних модов с другими версиями.
