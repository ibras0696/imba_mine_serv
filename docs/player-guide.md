# Инструкция для игроков

## 1. Установка Forge

- Версия Minecraft: **1.20.1**
- Forge: **47.4.10** (или актуальная из `.env`)

## 2. Установка модов

Самый простой способ:

```bash
bash git/scripts/install_client_mods.sh /путь/к/.minecraft/mods
```

На Windows удобнее запускать скрипт через Git Bash.

## 3. Подключение

- Основной сервер: `IP:PORT` (обычно `25565`)
- Shooter сервер: `IP:PORT` (обычно `25566`)

Если клиент пишет про несовпадение модов — обнови папку `mods` и убедись, что список соответствует серверу.
