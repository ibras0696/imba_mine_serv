# Обзор проекта

Этот репозиторий содержит готовую инфраструктуру для Minecraft Forge 1.20.1:
- два Forge-сервера (main и shooter) на одной сборке модов;
- отдельный vanilla-сервер без модов;
- Telegram-бот для управления сервером;
- автоматическое скачивание модов из Modrinth/CurseForge;
- шаблоны env и удобный Makefile.

## Что входит

- Docker Compose для трёх серверов и бота.
- Скрипты для скачивания модов и Forge installer.
- Документация по установке, настройке и работе.

## Структура репозитория

- `docker-compose.yml` — основной compose
- `docker/` — Dockerfile и артефакты Forge
- `env/` — примеры env
- `mods/` — моды и списки источников
- `data/` — данные серверов (не коммитятся)
- `bot/` — Telegram-бот
- `docs/` — документация

## Основные сценарии

- Поднять только бота: `make up-bot`
- Поднять основной сервер: `make up`
- Поднять все сервера: `make up-all`
- Управлять серверами через бота (кнопки + команды)
- Переключать миры и загружать карты через бота
