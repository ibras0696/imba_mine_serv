# ТЗ: несколько серверов + управление мирами через бота

## Цели

- Поднять два Forge-сервера (main и shooter) и один vanilla-сервер в одном проекте.
- Управлять всеми серверами через Telegram-бота.
- Дать возможность менять миры и загружать карты через бота.

## Серверы

- main: порт 25565
- shooter: порт 25566
- vanilla: порт 25567
- Данные раздельные:
  - `data/main` -> `/data`
  - `data/shooter` -> `/data`
  - `data/vanilla` -> `/data`
- Моды общие (Forge): `mods/server` -> `/data/mods`

## Управление мирами

- Переключение через `LEVEL_NAME`, `SHOOTER_LEVEL_NAME`, `VANILLA_LEVEL_NAME`.
- Список миров строится по директориям с `level.dat`.
- Операции:
  - показать список
  - переключить мир
  - создать новый мир (рандомный seed)
  - загрузить мир из ZIP или URL
- При изменении мира сервер останавливается и запускается заново.
- Перед заменой мира создается бэкап (хранить 2–3 последних).
- ZIP должен содержать `level.dat`, иначе ошибка.

## UI бота

- Кнопки для: запуск/остановка/перезапуск/статус/логи/моды/env.
- Отдельная кнопка выбора сервера (main/shooter/vanilla).
- Отдельное меню «Миры» с действиями.

## Очистка предметов

- Периодическая очистка только предметов (`@e[type=item]`).
- Таймер: 60 минут.
- Предупреждения: 30м, 20м, 10м, 5м, 1м, 30с и отсчет 5..1.
