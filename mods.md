# 1. RPG & Structures (данжи, генерация)

### Все моды ниже:

**обязаны быть на сервере**
**обязаны быть у клиента**

---

## 1. When Dungeons Arise

* **Тип:** Server + Client
* **Функция:** большие структуры, замки, крепости
* **Зависимости:** нет

## 2. Dungeons Plus

* **Тип:** Server + Client
* **Функция:** компактные RPG структуры
* **Зависимости:** нет

## 3. Structure Gel API

* **Тип:** Server + Client
* **Функция:** библиотека для структурных модов
* **Зависимости:** нет

## 4. YUNG's Better Series

Все модули:

* Better Dungeons
* Better Mineshafts
* Better Strongholds
* Better Nether Fortresses

**Тип:** Server + Client
**Зависимость:**

* YUNG's API (обязательно на сервере и клиенте)

## 5. YUNG's API

* **Тип:** Server + Client
* **Функция:** API для всех модов YUNG's

## 6. DivineRPG

* **Тип:** Server + Client
* **Функция:** большой RPG мод (измерения, мобы, боссы)
* **Зависимости:** нет
