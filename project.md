# Проект: Forge серверы + Telegram-бот

Этот репозиторий предназначен для запуска двух Forge серверов и управления ими через Telegram-бота.

## Цели

- Быстрое поднятие сервера с модами.
- Автоматическое скачивание модов.
- Удобное управление (бот, Makefile).
- Раздельные миры для main и shooter.

## Состав

- Docker Compose для двух серверов и бота.
- Скрипты для загрузки Forge и модов.
- Документация по запуску и эксплуатации.
